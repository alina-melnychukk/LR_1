Контрольні питання

1. Git- набір консольних утиліт, які відстежують і фіксують зміни в файлах; розподілена система керування версіями.
2. Стани : збережений у коміті, змінений, індексований.
3. Команда add . Додає необхідний файл до репозиторію.
4. Команда status. Перевіряє чи додані файли.
5. Команда commit. Фіксація стану сховища в певний момент часу.
6. Команда init. Ініціалізація Git у папці проекту.
7. Дані про зроблені зміни
8. Використовуючи команду git log.
9. Використовується для перемикання гілок та відгрузки їх вмісту в директорію. # LR_1
